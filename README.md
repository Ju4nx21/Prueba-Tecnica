# 🧮 Calculadora Web

## Descripción

Este proyecto consiste en una calculadora web desarrollada utilizando HTML, CSS y JavaScript. Su objetivo es proporcionar una interfaz sencilla, intuitiva y funcional que permita realizar operaciones matemáticas básicas de manera rápida y eficiente.

La aplicación fue desarrollada siguiendo los requisitos establecidos para la prueba técnica, implementando tanto funcionalidades esenciales como mejoras enfocadas en la experiencia del usuario.

---

## Funcionalidades Implementadas

### Operaciones matemáticas básicas

La calculadora permite realizar las siguientes operaciones:

- Suma (+)
- Resta (-)
- Multiplicación (×)
- División (÷)

### Manejo de números

- Números enteros.
- Números decimales.
- Números positivos.
- Números negativos mediante el botón (±).

### Funciones adicionales

- Botón **AC** para limpiar completamente la operación actual.
- Botón **←** para eliminar el último carácter ingresado.
- Botón **=** para obtener el resultado de la operación.
- Manejo de errores básicos durante el cálculo.

### Soporte de teclado

La aplicación permite ser utilizada tanto con el mouse como con el teclado.

Teclas soportadas:

| Tecla | Función |
|---------|----------|
| 0-9 | Ingreso de números |
| + | Suma |
| - | Resta |
| * | Multiplicación |
| / | División |
| . | Decimal |
| Enter | Calcular resultado |
| Backspace | Eliminar último carácter |
| Escape | Limpiar pantalla |

## Instalación y Ejecución

### Opción 1: Apertura directa

1. Descargar o clonar el proyecto.
2. Abrir el archivo `index.html` en cualquier navegador moderno.

### Opción 2: Visual Studio Code

1. Abrir la carpeta del proyecto en Visual Studio Code.
2. Instalar la extensión **Live Server**.
3. Hacer clic derecho sobre `index.html`.
4. Seleccionar **Open with Live Server**.
